## 命令行编辑中的快捷键

- `Ctrl + a`：移动到行首
- `Ctrl + e`：移动到行尾
- `Ctrl + k`：删除光标后的所有字符
- `Ctrl + u`：删除光标前的所有字符
- `Ctrl + w`：删除光标前的一个单词
- `Ctrl + y`：粘贴删除的字符
- `Ctrl + xx`：在光标和行首之间切换
- `Ctrl + h`：删除光标前的一个字符
- `Ctrl + i`：相当于 Tab 键
- `Ctrl + j`：相当于 Enter 键
- `Ctrl + m`：相当于 Enter 键

## 命令行回放快捷键

- `Ctrl + r`：搜索历史命令
- `Ctrl + g`：退出搜索
- `Ctrl + p`：上一条历史命令
- `Ctrl + n`：下一条历史命令

## 命令行控制快捷键

- `Ctrl + c`：中断当前命令
- `Ctrl + z`：挂起当前命令
- `Ctrl + l`：清屏
- `Ctrl + s`：暂停输出
- `Ctrl + q`：恢复输出

## 你为什么要使用这些快捷键

首先，使用快捷键肯定比鼠标更快，其次，还是请你多注意一下小拇指的感受罢，使用 Vim 多了，小拇指会非常地酸痛……
